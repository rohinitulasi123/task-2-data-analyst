# 📊 Data Analyst Task 2 – Power BI Dashboard (Superstore)

## 🧾 Overview

This project focuses on building an interactive Power BI dashboard based on Superstore sales data. The goal is to analyze key business metrics such as sales, profit, discount impact, and regional performance.

---

## 📁 Files Included

- `Superstore.csv` – The dataset used for analysis.
- `Task2_Report.pbix` – Power BI file with the full dashboard.
- `Task2_Report.pdf` – Exported static version of the report.
- `README.md` – Project overview and steps.

---

## 🛠 Tools Used

- Microsoft Power BI (Desktop)
- Data: Superstore (CSV file)

---

## 📊 Dashboard Features

The Power BI dashboard includes the following visuals:

1. **Sales by Category** (Bar Chart)
2. **Profit by Sub-Category** (Bar Chart)
3. **Discount vs Profit by Category** (Scatter Plot)
4. **Sales and Profit by Sub-Category** (Clustered Bar Chart)
5. **Sales by Region** (Pie Chart)
6. **Slicer for Region** (Interactive filter)

---

## 🎯 Key Insights

- Certain sub-categories have negative profit despite high sales.
- Discounts significantly affect profit in some categories.
- Region-wise sales performance varies, especially in the South.

---

## 📌 How to Use

1. Open `Task2_Report.pbix` in Power BI Desktop.
2. Use the **Region slicer** to filter the entire dashboard interactively.
3. View visual relationships between sales, profit, discounts, and categories.
